About

GoSeek Studio is a web implementation of GoSeek, an Open-Source Intelligence Apparatus.
GoSeek aims to centralize and expedite the reconnaisance experience by bringing together all the tools necessary to carry out an OSINT investigation.
It was also written so that developers could implement GoSeek into their applications with ease.
