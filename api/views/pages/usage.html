Tools

Under the tools section, you will find a list of different functions that you can use. The status of the tool is dictated by the link's color. To the right of the link, you will find a description that explains what each tool does and what kind of input it takes. In order to use the desired tool, just click on the link and fill in the necessary field(s). General access to API endpoints will soon be disabled.
