Tools GitHub Contribute

Below, you can find all of the tools that are currently available via the Omniscient API.

Tool Description
IP Lookup Retrieve detailed information on an IPV4 address.
Username Lookup Check hundreds of sites to see if there is an account with the specified username.
Discord Token Lookup Get detailed, sensitive information from an individual's Discord authorization token.
VIN Lookup Get a plethora of information about a vehicle's details using its Vehicle Identification Number
