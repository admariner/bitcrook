About

You are currently using a GUI/Web implementation of Cycull, an Open-Source Intelligence Apparatus. Cycull aims to centralize and expedite the reconnaisance experience by bringing together all the tools necessary to carry out an OSINT investigation. It was also written so that developers could implement Cycull into their applications with ease.
