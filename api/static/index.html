About

You are currently using a GUI/Web implementation of Bitcrook, an Open-Source Intelligence Apparatus. Bitcrook aims to centralize and expedite the reconnaisance experience by bringing together all the tools necessary to carry out an OSINT investigation. It was also written so that developers could implement Bitcrook into their applications with ease.
